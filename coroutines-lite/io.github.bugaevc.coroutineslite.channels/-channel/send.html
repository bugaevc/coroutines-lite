send

open suspend override fun send(element: E)
Content copied to clipboard

Send the element through this channel.

A call to this method will only suspend if there's no receiver waiting to receive, the buffer is full, and the buffer overflow handling strategy is set to BufferOverflow.SUSPEND.

If the channel is already closed for send, a call to this method will throw the cause exception.
