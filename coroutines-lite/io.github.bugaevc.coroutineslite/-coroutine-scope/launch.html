fun launch(context: CoroutineContext = EmptyCoroutineContext, block: suspend CoroutineScope.() -> Unit): Job
Content copied to clipboard

Launch a new asynchronous job in this scope, running block.

This method is intended for fire-and-forget launching of jobs. If you need to execute a block of code asynchronously and later retrieve its result, use async instead.

You can optionally specify a context for the new job to run in. For instance, you can write:

launch(Dispatchers.IO) {
    // ...
}

instead of the longer:

launch {
    withContext(Dispatchers.IO) {
        // ...
    }
}
