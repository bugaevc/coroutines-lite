Flow

interface Flow<out T>
Content copied to clipboard

An asynchronous sequence of values.

Flow is the coroutine counterpart to Sequence.

The two primary ways to build a flow are with the flow and channelFlow builders, or by using a MutableStateFlow. A flow can be transformed using the various flow transformers such as map, filter, takeWhile, zip, debounce, merge, and others. Finally, a flow can be collected by calling the collect method defined by this interface.
