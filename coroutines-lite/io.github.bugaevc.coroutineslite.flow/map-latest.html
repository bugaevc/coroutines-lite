fun <T, R> Flow<T>.mapLatest(transform: suspend (T) -> R): Flow<R>
Content copied to clipboard

Run this flow concurrently with applying transform to its values.

If the flow emits a new value before transforming the previous value is completed, transforming the previous value is cancelled, and the value is skipped.
