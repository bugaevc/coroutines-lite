fun <T> Flow<T>.filter(predicate: suspend (T) -> Boolean): Flow<T>
Content copied to clipboard

Filter this flow, leaving only elements that satisfy the given predicate.

This will return a new flow which applies predicate to each item of the original flow, and only re-emits those items that satisfy the predicate.
