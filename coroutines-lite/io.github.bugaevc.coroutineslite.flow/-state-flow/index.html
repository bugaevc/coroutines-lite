StateFlow

interface StateFlow<out T> : Flow<T>
Content copied to clipboard

State flow stores a value and emits when the value is updated.

You can additionally always get the current value directly, without suspending. The actual implementation of the state flow interface, MutableStateFlow, also lets you modify the value.

Multiple collectors can collect the same state flow concurrently. They will all see an emission of the new value when the value is modified.

A state flow always behaves as if distinctUntilChanged and conflate modifiers had been applied; it will never emit several equal values is a row, and will always emit the latest value.
